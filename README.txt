He creado un sistema de partículas en 3D personalizable siguiendo las pautas dadas en el PDF.

El usuario puede cambiar tanto el angulo inicial, como la velocidad, tiempo de vida, y gravedad si quiere.
Cada partícula tiene un angulo inicial, velocidad y tiempo de vida aleatorios al nacer.
Cada vez que una partícula se destruye se genera una partícula nueva, al inicio el usuario puede indicar cuantas partículas quiere que salgan.
